#####2. redis

######1.创建服务器，连接本地步骤：

  1. 检查是否安装成功：redis-cli –version

  2. 查看是否有启动的进程：netstat -nap | grep redis

  3. 如果有的话就杀进程：kill 进程号

  4. 检查版本：redis-server –version

  5. 启动服务器：redis-server

  6. 加上&让命令在后台运行：redis-server & 先不要按回车，可以把显示的内容加到日志文件里面

    redis-server > redis.log

    改端口号：redis-server –port 新端口号 redis.log &

    加口令：redis-server –requirepass 口令密码 > redis.log &

  7. 端口号：6379 (不能公开此端口,因为会被攻击非要公开的话要加口令) | kill 进程号 杀进程(PID)

  8. 服务器启动好，就用客户端去连接

    连自己本机6379：redis-cli 默认开了16个数据库 用select来切换底层数据库

    连接他人：redis-cli -h 指定主机IP地址 —- 如果这台主机没有开6379端口是连不上的

    连接他人指定端口：redis-cli -h 指定主机 -p 指定端口

  9. 连接好后验证：输入ping，会回应PONG，如果服务器设置了口令密码，必须要先写：auth 口令密码

  10. 查看进程号：pgrep redis

  11. 查看后台运行的任务：jobs

  12. 把后台移到前台运行：fg %1

  13. 再移到后台Ctrl + z 后台是停止的

  14. 把任务放在后台运行bg %1

  15. 把输出输出到文件中去：redis-server > redis.log 2> redis-error.log &

  16. redis的基本测试(查看你的redis一秒可以做多少次的读写操作，查看你的redis的性能好不好)：redis-benchmark

######3. set字符串类型

set 字符串形式的键值对

  1. 查看数据类型：type 键

  2. 可以查看底层数据类型：

  3. 创建一个键值对：set 键 值

    例如1：set username candy

    例如2：set password 123456

  4. 可以看到你放的所有键值对：keys *

  5. 取出来：get [键值]

  6. 设置过期时间为30s：127.0.0.1:6379> set username candy ex 30

  7. 查看还有多少秒过期：ttl 键值

    -1永不过期，-2已经过期

  8. 127.0.0.1:6379> expire email 100 设置过期日期

  9. 127.0.0.1:6379> del password 删除键值对

  10. 127.0.0.1:6379> del password 查看键存不存在 0表示不存在，1表示存在

  11. 可以做投票 incr 键

  12. flushdb - 清除当前数据库的数据

  13. flushall - 清除所有的数据库的数据

  14. randomkey - 随机取键

  15. dbsize - 查看当前数据库一共放了多少键值对

  16. mset - 可以一次性加多个键值对

  17. mget - 可以一次性将多个键的值取出来

  18. 关闭服务 shutdown

  19. 关闭 quit

  20. save 手动保存数据

  21. bgsave 在后端保存数据

######4. 哈希存储

哈希存储 - 性能非常好的存储方案 - set

obj — 哈希函数 —> 哈希码（散列码）—> 对象在内存中存储的位置

文件 —>MD5 / SHA1 / SHA256 —> 哈希码（数字指纹 / 数字签名）

哈希表，特别适合保存对象，相当于嵌套了字典

  1. hget
  2. 凡是以h开头都是哈希类型的
  3. 查看所有：hgetall
  4. 删除：hdel
  5. 查看所有的键：hkeys 对象名
  6. 查看所有的值：hvals 对象名

######5. 列表(可以保存多个元素，可以放重复元素)

  1. 列表可以用于消费者和

  2. 从左边放：lpush

    例如：lpush numbers 10 20 30

  3. 取全部：lrange numbers 0 -1

    “30”

    “20”

    “10”

  4. 取元素：lpop numbers

    “30”

  5. 从右边放：rpush

    例如：rpush numbers 10 20 30

  6. 取元素：rpop numbers

    “30”

    取元素：lpop numbers

    “10”

######6. 集合(可以放多个元素，去重)

  1. 有序集合可以做排行榜

  2. 添加元素：sadd 集合名 多个元素

    sadd set1 10 20 20 10 40 30

  3. 查看：smembers set1

    “10”

    “20”

    “40”

    “30”

  4. 查看集合有几个元素：scard set1

  5. 查看某元素是否在集合里面：sismember set1 要查询的元素

    返回0，则为假

    返回1，则为真

  6. 集合可以求交集、并集、差集

    交集：sinter 集合1 集合2

    并集：sunion 集合1 集合2

    差集：sdiff 集合1 集合2

######7. 有序集合

  1. 可以做实时排行榜

  2. 排序：zadd phb 100 yuting 50 luohao 900 wanghaifei 3000 xiaoshirong

  3. 查看（从小到大）：zrange phb 0 -1

    “luohao”

    “yuting”

    “wanghaifei”

    “xiaoshirong”

  4. 查看（从大到小）：zrevrange phb 0 -1

  5. 给元素增加票数 ：zincrby phb 5000 yuting

    返回：5100
